# A Era Artificial — Site de divulgação

Pequena página estática para divulgar o livro de ficção científica "A Era Artificial".

- Arquivos principais: `index.html`, `estilo.css`.
- Imagens/placeholder em `assets/`.

Como usar:

1. Abra `index.html` em um navegador.
2. Edite o conteúdo conforme necessário (título, sinopse, links de compra).

Observações:

- Este repositório foi renomeado para melhor refletir o conteúdo do projeto.
- Imagens de capa e autor são placeholders SVG.
